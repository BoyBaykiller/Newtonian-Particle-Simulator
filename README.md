### **KeyBoard:**
* W, A, S, D => Movment
* E => Toggle cursor visibility
* T => Toggle simulation
* V => Toggle VSync
* F11 => Toggle fullscreen
* LShift => Faster movment speed
* LControl => Slower movment speed
* Esc => Close

### **Mouse:**
* LButton => Set point of mass
